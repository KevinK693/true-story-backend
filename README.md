# True Story - Backend

API REST du jeu collaboratif True Story. Gère les parties, 
les joueurs, les tours et la communication avec l'IA Gemini.

## 🛠️ Technologies
- Node.js
- Express.js
- MongoDB
- API Gemini (Google AI)

## ⚙️ Installation
```bash
git clone https://github.com/KevinK693/true-story-backend
yarn install
yarn dev
```
